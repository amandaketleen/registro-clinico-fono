## 💻 Projeto: 🩺 Sistema de Registro Clínico Fonoaudiológico

Sistema web desenvolvido para organização e registros clínicos de pacientes em atendimentos fonoaudiológicos.

O projeto permite cadastrar pacientes, registrar informações clínicas e anexar documentos importantes relacionados ao acompanhamento terapêutico.

---

## 🚀 Funcionalidades

- Login de acesso ao sistema
- Cadastro de pacientes
- Visualização da ficha do paciente
- Anamnese automática (adulto ou infantil)
- Registro de evolução clínica por data
- Upload de devolutivas
- Upload de encaminhamentos
- Armazenamento de dados utilizando LocalStorage

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
- Git e Github
